Message Pad

Visit MessagePad.

This personal project is my most recent experience. It is a real-time chat application involving NodeJS, ExpressJS and Socket.io. A user has the facility to enter his/her username and start chatting. A user can also open multiple instances of this application in a browser's tabs. The app also displays the number of users online.

Additions I plan on making - (Giving it an IRC feel.)

  • Ability to see who is online and who has disconnected.
  • Add "user is typing.." functionality.
  • Add support for nicknames.

Contacts App

Visit Contacts App.

This is a contacts application where a user can save his/her contacts in Firebase. It's client side framework is AngularJS. It supports CRUD functionality. Just to try something different with this project, I've given Foundation a go with this one! I would say I did not feel there were very many differences between Foundation and Bootstrap. A lot of the syntax specifications are similar so I learned it pretty quickly!

Additions I plan on making -

  • User login functionality.
  • Add photo for the contact.
  • Add birthday reminders!

Twitter Bot

This project utilizes NodeJS and Twitter API along with the twit REST API client. In the demo video, the user @rindu23_gemini tweets to @saurabh_rana whose account is locked with the twitter bot. When rindu23_gemini tweets to @saurabh_rana, the bot automatically replies "Thank you for tweeting me" to rindu23_gemini. This is an automatic reply bot locked to a specific user account.
